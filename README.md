1. **Clone the repo:**
   ```sh
   git clone https://github.com/your-org/ketryon.com.git
   cd ketryon.com
   ```
2. **Install dependencies:**
   ```sh
   pnpm install
   ```

4. **Open [http://localhost:3000](http://localhost:3000) in your browser.**
